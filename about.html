I'm eager to learn.

"You should learn more" is what my Dad always said. He was right. I decided to give my web skills an upgrade.
I started out small. Codecademy, FreeCodeCamp ... Those kind of things.

But that got boring pretty quickly and Udemy was my next stop. Some courses you should try:

  • The Web Developer Bootcamp - This was the first course I bought on Udemy.
    And you can learn so much from it: jQuery, Express, REST (hang that table above your bed), Mongoose ...
    It's worth its length in gold!
  • Object-oriented programming in Javascript - "Small" course with little practical stuff here.
    But there's nobody who can explain "prototypical inheritance" as good as Mosh!

Then I spent some time learning the basics of SQL and MongoDB through books and experimentation.
Around that time, I decided to get serious with my transformation from marketing to IT. So I set my eyes on a 1,5 year course at Howest named Blockchain Developer & Architect. A sneak preview of course content:

  • Hashing and encryption algorithms and how they work under the hood.
  • How to set up a Hyperledger Fabric network.
    If you're looking for detailed articles about Hyperledger, Ledger Tech is where you need to be!
  • Working with an Arduino Uno and integrating it into a blockchain network (best course ever!)
  • More general topics such as security management, IT law, BPM ...
